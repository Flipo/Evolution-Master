Das Jahr 2012

Das mobile Web ist auf dem Vormarsch. Immer mehr Internetseiten werden auf die Bildschirmgrößen von Smartphones und Tablets angepasst. Um die immer größer werdende Masse an Informationen verarbeiten zu können steht ein klarer Aufbau, gute Lesbarkeit und einfache Benutzung im Mittelpunkt des aktuellen Webdesign.
